Instructions:
* Make sure “client.py”, “server.py”, “packet.py”, “checksum.py”, “rdt22.py”, and the image file to be sent (“img2.jpg” is provided) are all saved in the same directory.
* When within the directory using the terminal run “./server.py”, and in a second terminal run “./client.py”
* In the second terminal in which “./client.py” was run, input the requested image name including the extension format of the image. (eg. “img2.jpg”)
* Enter the name that the new image will be saved under, also including the file extension. (eg. “imgServer.jpg”)
*  Open the new image saved under the name input in the previous step to verify absence of packet loss.
