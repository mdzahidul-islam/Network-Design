Environment
* Operating System: Ubuntu 18.04
* Language: Python 3.6.9

Instructions:
* Make sure “server.py”, “client.py”, “tcp.py”, “checksum.py”, “packet.py” and the image file to be sent (“img2.jpg” is provided) are all saved in the same directory.
* When within the directory using the terminal run “./server.py”, and in a second terminal run “./client.py”
* In the first terminal in which “./receiver.py” was run, input the image name the file will be saved under including the extension format of the image. (eg. “img2.jpg”)
* In the second terminal in which “./sender.py” was run, input the requested image name including the extension format of the image. (eg. “img2.jpg”)
* Open the new image saved under the name input in the previous step to verify absence of packet loss.
